# misc

Minimal Instruction Set Computing - a program that runs a super tiny ISA.

This is written in C, I call it the _minimal instruction set_ for computing, even though I don't think it actually is. This program can interpret plaintext assembler written in MISC and run it locally. I thought of this randomly while in a deep rabbit hole about older computer architectures and pondering a past uni course I did.

Why C? I know C, and learning a new language like Rust or something would be a little much for what is ultimately an experiment. The full instruction set as well as how its encoded and how the _MISC CPU_ is designed are in [another document!](.)
